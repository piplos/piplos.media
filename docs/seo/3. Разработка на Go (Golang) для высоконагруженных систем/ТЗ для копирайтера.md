### Блок 2: Почему Go — это стандарт для современного бэкенда?
*   Нужно описать 3 ключевых фактора:
    1.  **Невероятная скорость:** Go работает почти так же быстро, как C++, что критично для систем с миллионами пользователей.
    2.  **Эффективная многопоточность:** Уникальная модель горутин позволяет обрабатывать тысячи запросов одновременно при минимальном потреблении ресурсов.
    3.  **Стабильность и простота:** Код на Go легче поддерживать и масштабировать, что снижает риски для долгосрочных проектов.
